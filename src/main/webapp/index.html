Smart Campus API

Base path: /api/v1

Discovery

  • GET /api/v1 - API discovery document

Rooms

  • GET /api/v1/rooms - list all rooms
  • POST /api/v1/rooms - create a room
  • GET /api/v1/rooms/{roomId} - get a room by id
  • DELETE /api/v1/rooms/{roomId} - delete a room

Sensors

  • GET /api/v1/sensors - list all sensors
  • GET /api/v1/sensors?type={type} - filter sensors by type
  • POST /api/v1/sensors - create a sensor

Sensor Readings

  • GET /api/v1/sensors/{sensorId}/readings - list readings for a sensor
  • POST /api/v1/sensors/{sensorId}/readings - add a reading to a sensor
